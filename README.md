## About

Software Engineer specializing in **backend architecture**, **AI-powered systems**, and **business automation**. I build scalable products that balance speed, clarity, and control.

**What I do:**
- Design scalable REST APIs and backend architectures
- Integrate LLMs into production workflows with validation and guardrails
- Build multi-tenant SaaS systems with strong isolation and audit trails
- Deploy full-stack products with Django, Next.js, PostgreSQL, and AWS

**Tech I work with daily:**
`Python` `Django` `FastAPI` `PostgreSQL` `Next.js` `React` `OpenAI` `LangChain` `Docker` `AWS`

## AI & LLM Engineering

I design AI systems that interact with real-world business workflows. My focus is on building **production-ready** AI solutions, not just prototypes.

| Area | Description |
|------|-------------|
| **RAG Architectures** | Retrieval-augmented generation systems for enterprise knowledge bases |
| **AI Agents** | Autonomous agents with tool use and multi-step reasoning |
| **LLM Integration** | OpenAI, LangChain, and custom model deployment |
| **Vector Databases** | Chroma, FAISS, and vector search optimization |
| **Prompt Engineering** | Systematic prompt design with validation and guardrails |
| **Human-in-the-Loop** | Workflows combining AI efficiency with human oversight |

> **Philosophy:** Ship AI that works in production — with validation, guardrails, and human oversight.
